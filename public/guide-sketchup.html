3. Construction etape par etape

Etape A — La boite du camion

A

1. R Rectangle au sol → tape 2.045,4
2. P Push/Pull vers le haut → tape 1.98
3. Selectionne tout → clic droit → Make Group
4. Supprime la face du dessus et la face arriere (pour voir l'interieur, vue cutaway)

Astuce : Pour la vue cutaway comme les images Le Van Migrateur, supprime le toit + un mur lateral. Tu peux aussi utiliser Section Plane (menu Tools) pour couper proprement.

Etape B — L'appareil IRM

B

1. R Rectangle → tape 0.82,1
2. P Push/Pull → tape 1.5
3. Positionne dans le coin avant-conducteur (origine)
4. Pour le bore : dessine un cercle (diam 0.52m) au centre de la face arriere de l'IRM, Push/Pull vers l'interieur
5. Make Group

OU mieux : cherche un modele dans le 3D Warehouse (voir section 5)

Etape C — Table patient

C

1. R Rectangle → 0.75,1.82
2. P Push/Pull → 0.05 (epaisseur plateau)
3. M Move → aligne le bore de l'IRM (Y=1.0, centree X sur l'IRM)
4. Ajoute 4 pieds cylindriques (hauteur 0.725m)
5. Make Group

Etape D — Armoire electrique

D

1. R Rectangle → 0.762,0.762
2. P Push/Pull → 0.762
3. M Move vers la paroi PASSAGER → position X=1.283, Y=2.119
4. Make Group

Etape E — Rangement

E

1. R Rectangle → 0.5,0.6
2. P Push/Pull → 0.8
3. M Move → position X=1.545, Y=3.1, contre la paroi passager
4. Make Group

Etape F — Zone de degagement

F

1. Dessine un arc de 90 degres depuis le coin interieur de l'IRM (X=0.82, Y=1.0)
2. Rayon = 1.0m, du mur conducteur vers le mur passager, de l'IRM vers l'arriere
3. Laisse-le comme trait au sol (pas besoin de l'extruder)
4. Couleur rouge transparent

5. 3D Warehouse — Modeles a telecharger

Dans SketchUp, clique sur l'icone 3D Warehouse (petit entrepot) et cherche ces termes :

Machine IRM Cherche : MRI machine, MRI scanner, Siemens MAGNETOM
Table medicale Cherche : MRI patient table, medical stretcher, hospital bed
Armoire electrique Cherche : electrical cabinet, server rack, electrical panel
Camion cube Cherche : box truck, cube van, Ford Transit 350
Armoire rangement Cherche : medical cabinet, storage cabinet
Personnage (echelle) Cherche : person standing pour donner l'echelle
Astuce : Quand tu telecharges un modele du 3D Warehouse, utilise S (Scale) pour l'ajuster aux bonnes dimensions, et T (Tape Measure) pour verifier.

6. Vue cutaway (comme les images)

Pour obtenir la vue isometrique avec l'interieur visible (style Le Van Migrateur) :

1
Section Plane : Menu Tools → Section Plane. Place le plan de coupe horizontalement pour couper le toit, ou verticalement pour couper un mur.
2
Camera : Menu Camera → Parallel Projection pour une vue isometrique (pas de perspective). Puis Camera → Standard Views → Iso.
3
Style : Va dans Styles et choisis un style avec ombres. Active les ombres dans Shadows pour plus de realisme.
4
Export : File → Export → PNG en haute resolution (4000px+ de large).

7. Plan de reference (vue de dessus)

Utilise ton plan 2D comme guide : Ouvre plan-irm.html dans un autre onglet et garde-le visible a cote de SketchUp.

Tu peux aussi faire un screenshot du plan de dessus et l'importer dans SketchUp comme image de reference au sol (File → Import → Image).

8. Checklist finale

  • Boite camion : 2.045 x 4.000 x 1.980m
  • IRM : coin avant-conducteur, 0.82 x 1.0 x 1.5m
  • Table : alignee bore, 0.75 x 1.82m a 0.75m du sol
  • Armoire elec. : cote PASSAGER (oppose IRM), 0.762m cube
  • Rangement : cote passager, derriere armoire, 0.6 x 0.5 x 0.8m
  • Zone degagement : 90deg, 1.0m rayon, depuis coin IRM
  • Materiaux : blanc IRM, inox table, vinyle sol, metal armoire
  • Vue : Section Plane + Parallel Projection + Shadows
  • Export : PNG haute resolution pour l'appel d'offres
